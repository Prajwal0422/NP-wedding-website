# Wedding Assets

## Required Files

### Audio Files
- `carnatic-music.mp3` - Background Carnatic music (optional)
- `bell-chime.mp3` - Temple bell sound for page load (optional)

### PDF
- `invitation.pdf` - Downloadable wedding invitation card

## Instructions

1. Add your Carnatic music file as `carnatic-music.mp3`
2. Add temple bell sound as `bell-chime.mp3` (optional)
3. Add your PDF invitation as `invitation.pdf`

## Notes
- Music files should be optimized for web (compressed, reasonable file size)
- PDF should be under 5MB for easy download
